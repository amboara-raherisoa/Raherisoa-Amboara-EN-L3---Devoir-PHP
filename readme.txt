*** Création de la base de données ***

 (SGBD utilisé: MySQL)


- aller dans le dossier "mada2rouesMVC/db/"

- exécuter le script "mada2roues.sql" (par exemple via phpMyAdmin)

pour créer la base de données "m2ramboara"


*** Installation du projet ***
Copier le dossier "mada2rouesMVC/" vers la racine du serveur local
(exemple: si on utilise Wampserver64, on copie le dossier vers C:\wamp64\www\)

*** Indications ***
- Pour accéder à la page d'accueil, aller à l'adresse:
	http://localhost/mada2rouesMVC/Web/

- Pour accéder à l'espace d'administration:
 . aller à l'adresse:
	http://localhost/mada2rouesMVC/Web/backend.php
 . pseudo : admin
   mot de passe: mdp
 . cliquer sur le menu Admin
